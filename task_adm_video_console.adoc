= Accédez à la console vidéo ONTAP Select
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Vous pouvez accéder à la console vidéo de la machine virtuelle de l'hyperviseur sur laquelle ONTAP Select est exécuté.

.À propos de cette tâche
Vous pourriez avoir besoin d'accéder à la console de la machine virtuelle pour résoudre un problème ou lorsque cela vous est demandé par le support NetApp.

.Étapes
. Accédez au vSphere client et connectez-vous.
. Accédez à l'emplacement approprié dans la hiérarchie pour localiser la machine virtuelle ONTAP Select.
. Faites un clic droit sur la machine virtuelle et sélectionnez *Ouvrir la console*.
