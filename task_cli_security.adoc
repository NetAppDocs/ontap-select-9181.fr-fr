= Modifier le mot de passe de l'administrateur Deploy

Vous pouvez modifier le mot de passe du compte administrateur de la machine virtuelle Deploy selon vos besoins à l'aide de l'interface de ligne de commandes.

.Étapes
. Connectez-vous à l'interface de ligne de commande (CLI) de l'utilitaire Deploy en utilisant le compte administrateur.
. Changer le mot de passe :
`password modify`
. Répondez à toutes les invites en fonction de votre environnement.
